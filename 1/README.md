# C/C++ starter
## Author
Makarevich Dmitriy  821702(7-th variant) 
This project contains `C/C++` starter configuration.
To start using it put source files and headers into `src` folder.

## How to use

### Generate Makefile:
```bash
cmake .
```
### Build project:
```bash
make
```

### Run project:
```bash
./bin
```
